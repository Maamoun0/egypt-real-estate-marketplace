## 🌟 نظرة عامة (Overview)

**إمكان (EMKAN)** هي منصة عقارية رائدة صُممت خصيصاً لتلبية احتياجات السوق المصري الراقي. تركز المنصة على تقديم أرقى العقارات في المناطق الأكثر طلباً مثل **القاهرة الجديدة، الشيخ زايد، والساحل الشمالي**. 

تم تصميم وتطوير هذا المشروع برؤية تهدف إلى سد الفجوة بين الفخامة العقارية والحلول الرقمية المتطورة، حيث يجمع بين واجهة مستخدم (UI) مبهرة وتجربة مستخدم (UX) سلسة تضمن وصول العميل إلى منزله المثالي بأقل مجهود.

## 🚀 الميزات التقنية والوظيفية (Key Features)

### 💎 تجربة مستخدم استثنائية
- **🌍 دعم ثنائي اللغة (Bilingual Mastery):** نظام متطور للتبديل بين العربية والإنجليزية مع تعديل تلقائي للعناصر (RTL/LTR) والخطوط والمسافات بما يخدم جمالية كل لغة.
- **🎨 نظام تصميم مخصص (Design System):** لوحة ألوان ملكية تعتمد على الأزرق الداكن والذهبي، مع دعم كامل للوضع الليلي (Dark Mode) الذي يبرز فخامة الصور المعمارية.
- **📱 تصميم متجاوب (Fluid Responsiveness):** تجربة تصفح مثالية على كافة أحجام الشاشات، من الهواتف الذكية إلى شاشات الـ 4K.

### 🔍 محرك بحث وتصفية متقدم
- **Interactive Filtering:** إمكانية البحث والتصفية حسب الموقع، السعر، النوع (فلل، شقق، أراضي)، والغرض (بيع/إيجار) بشكل لحظي.
- **Instant Search:** نتائج فورية وتفاعل سلس دون الحاجة لإعادة تحميل الصفحات.

### 💬 تواصل وحجز مباشر
- **One-Click Communication:** أزرار مدمجة للتواصل المباشر عبر WhatsApp أو الاتصال الهاتفي لضمان سرعة التحويل (Conversion).
- **Property Listing System:** نموذج احترافي للملاك والمطورين لإضافة عقاراتهم مع أدوات للتحقق من جودة البيانات.

## 📂 هيكل المشروع (Project Structure)

```text
├── public/                # الواجهة الأمامية والملفات العامة
│   ├── css/               # ملفات التنسيق والأنماط
│   ├── images/            # الصور واللقطات التوضيحية
│   └── js/                # منطق الواجهة الأمامية (Frontend Logic)
├── src/                   # الكود المصدري للواجهة الخلفية
│   ├── controllers/       # إدارة الطلبات والمسارات
│   ├── services/          # منطق العمل الأساسي (Business Logic)
│   ├── models/            # نماذج وهياكل البيانات
│   └── utils/             # دوال وأدوات مساعدة
└── tsconfig.json          # إعدادات لغة TypeScript
```

## 💻 دليل التشغيل المحلي (Quick Start)

للحصول على نسخة من المشروع وتشغيلها على جهازك، اتبع الخطوات التالية:

1. **تحميل التبعيات:**
   ```bash
   npm install
   ```

2. **بدء التشغيل في بيئة التطوير:**
   ```bash
   npm run dev
   ```

3. **رابط المعاينة:**
   افتح [http://localhost:3000](http://localhost:3000) في متصفحك.
